# News Search

This skill searches for recent news on any topic using web search.

## Parameters

- **topic**: News topic to search for (e.g., "technology", "politics", "sports")

## Usage Examples

```
Get the latest news about AI
```

```
Search for today's news on climate change
```

```
What's the latest news about Tesla?
```

## How It Works

1. Performs a web search with the topic + "news today"
2. Returns the most relevant and recent results
3. Provides a summary of current events

## Notes

- Results are based on web search, may include various sources
- For real-time news, consider checking specific news websites
- Quality depends on search results and recency
